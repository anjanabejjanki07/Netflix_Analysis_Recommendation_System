# 🎬 Netflix Interactive Analytics & Recommendation System

An interactive web application that allows users to upload their own Netflix dataset, explore insights, and get movie recommendations using Machine Learning.

---

## 🌐 Live Demo

👉 Try the app here: **http://localhost:8501**

---

## 🚀 Features

* 📁 Upload your own CSV dataset
* 🔍 Search movies and TV shows
* 🎭 Filter by genre, country, and type
* 📊 Visualize top genres and trends
* 🌍 Analyze content distribution by country
* 🤖 Get personalized movie recommendations

---

## 🛠️ Tech Stack

* Python
* Streamlit
* Pandas
* Scikit-learn

## ▶️ Run Locally

1. Clone the repository:

```
git clone https://github.com/anjanabejjanki07/Netflix_Analysis.git
```

2. Navigate to the folder:

```
cd Netflix_Analysis
```

3. Install dependencies:

```
pip install -r requirements.txt
```

4. Run the app:

```
streamlit run app.py
```

---

## 📊 Dataset

Users can upload their own Netflix dataset (CSV format).
Recommended columns:

* title
* listed_in (genre)
* country
* description
